# Language Detector

Detect the language of any text. Character n-gram frequency analysis evolved from Wikipedia corpora. 50 languages, 95%+ accuracy on 20+ chars.

Part of [FreeAgentStore](https://freeagentstore.online) — AI tools that run free in your browser.

## Install

```bash
npm install @freeagentstore/language-detector
```

## Usage

```ts
export { detectLanguage, getFlag } from "@freeagentstore/language-detector";
```

## ESM (zero install)

```html
<script type="module">
  import { ... } from 'https://freeagentstore.online/pkg/language-detector/index.js'
</script>
```
